# CommunityFit Gym Website

## Project Overview

CommunityFit Gym Website is a static front-end website created for the User Centric Frontend Development Milestone Project.

The purpose of the website is to present useful information for current and potential gym members in a clear, friendly, and easy-to-use format. The site focuses on helping users quickly understand what the gym offers, including classes, trainers, membership information, and contact details.

## User Goals

The website is designed to help users:

- learn about the gym and its community-focused environment
- explore available fitness classes
- view trainer information
- understand membership options
- find contact details and opening information

## Site Goals

The main goals of the project are to:

- create a clear and accessible informational website
- present static content in an organised way
- provide intuitive navigation between pages
- demonstrate user-centred design planning through wireframes and structure
- build the site using HTML and CSS

## Features

The website includes:

- a home page introducing the gym
- a classes page showing available fitness classes
- a trainers section/page
- a membership section/page
- a contact section/page
- navigation menu for moving between key areas of the site
- supporting project documents in the assets folder

## Pages / Structure

The project is organised around the following main content areas:

- Home
- Classes
- Trainers
- Membership
- Contact

## Design and Planning

Before development, the website structure and layout were planned in Figma.

The `assets` folder currently contains supporting planning documents:

- `Brief & Goals.pdf`
- `Sitemap IA.pdf`
- `Wireframes.pdf`

These documents show the initial project brief, information architecture, and wireframe layouts used to guide development.
