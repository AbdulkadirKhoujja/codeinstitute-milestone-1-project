CommunityFit Gym Website
Project Overview

CommunityFit Gym Website is a static front-end website created for the User Centric Frontend Development Milestone Project.

The purpose of the website is to present useful information for current and potential gym members in a clear, friendly, and easy-to-use format. The site focuses on helping users quickly understand what the gym offers, including classes, trainers, membership information, and contact details.

User Goals

The website is designed to help users:

learn about the gym and its community-focused environment

explore available fitness classes

view trainer information

understand membership options

find contact details and opening information

Site Goals

The main goals of the project are to:

create a clear and accessible informational website

present static content in an organised way

provide intuitive navigation between pages

demonstrate user-centred design planning through wireframes and structure

build the site using HTML5 and CSS3

Features

The website includes:

a Home page introducing the gym and its community-focused environment

a Classes page showing available fitness classes

a Trainers page presenting gym trainers

a Membership page outlining membership options

a Contact page providing contact information and opening hours

a navigation menu allowing users to move between all sections of the website

clear layout and structured presentation of static information

Pages / Structure

The project is organised around the following main content areas:

Home

Classes

Trainers

Membership

Contact

Each page focuses on presenting specific information to help users easily understand the services offered by the gym.

Design and Planning

Before development, the website structure and layout were planned using Figma.

The assets folder contains supporting planning documents created during the design phase:

Brief & Goals.pdf

Sitemap IA.pdf

Wireframes.pdf

These documents demonstrate the initial project brief, the information architecture of the site, and the wireframes that guided the final layout and design.

Images

All images used on the website were obtained from Pexels, which provides free stock photography.

Images were selected to reflect a welcoming and community-focused gym environment.

All images included on the site have alt text descriptions, and the selected images were chosen to match the meaning of the alt text. This helps improve accessibility and ensures that screen readers can accurately describe the visual content of the website.
